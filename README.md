# Hostel Management System

## 📖 Project Overview
The Hostel Management System is a web-based application developed using Java, JSP/Servlets, Apache Tomcat, and MySQL.  
It helps in managing hostel operations such as student registration, room allocation, and record management.

---

## 🛠 Technologies Used
- Java
- JSP / Servlets
- Apache Tomcat Server
- MySQL Database
- HTML, CSS

---

## 📂 Project Structure
- Java Source Code (Eclipse Dynamic Web Project)
- JSP Pages
- Servlet Classes
- MySQL Database File (hostel_database.sql)
